No CSS nós possuimos os seletores por ID

Olá tudo bem?

  • Seletores por Tipo: Busca elementos por uma tag HTML especifica
  • Seletores por ID (#): Busca elementos por atravez do atributo "id"
  • Seletores por Class (.): Busca elementos por atravez do atributo "class"
  • Seletores por Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona alementos que possuem um atributo especifico em nosso documento HTML e conseguimos também atributos com um valor especifico
